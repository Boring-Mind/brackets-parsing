# Parse brackets
This is an assignment from one of the tech companies.

You will have a string of four kind of brackets: (), [], {}, <>. You will have to check if a provided string is valid or not.

Valid strings:
- "([<>])"
- "()[]<{}>"
- "[<>()[(<>)]]"

Invalid strings:
- "([)]"
- "[[]"
- "<(}>"
- ">"
